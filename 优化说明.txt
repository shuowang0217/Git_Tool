Git-Tool v1.5.1 更新说明

本次修复：
1. 同步 README 版本号，由 Git-Tool v1.5 改为 Git-Tool v1.5.1。
2. 同步 build.bat 打包名称，输出 exe 改为 dist\Git-Tool-v1.5.1.exe。
3. 保留 Git-Tool-v1.5.1.spec，删除旧的 Git-Tool-v1.5.spec，避免误用旧版本配置。
4. 主程序版本号为 APP_VERSION = "1.5.1"。
5. 默认主分支逻辑继续使用 master。

保留上一版优化：
1. 修复“一键保存并上传”不会弹出版本说明输入框的问题。
2. 普通“保存版本”和“一键保存并上传”统一使用同一个版本说明输入逻辑。
3. 一键保存并上传时默认说明仍为 save: 时间，但允许用户手动修改。
4. 用户在版本说明窗口点击取消或输入为空时，会停止本次保存上传，不会自动提交默认说明。
5. 初始化仓库默认分支由 main 改为 master。
6. current_branch() 兜底返回由 main 改为 master。
7. 初始化完成提示中增加当前主分支名称。
